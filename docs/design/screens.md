# 画面設計書（screens）

**Auto Video AI Agent**

本ドキュメントは MVP の画面構成と、各画面における UI・機能要件を定義する。

---

# 🎨 1. 画面一覧（MVP）

| 画面ID | 画面名 | 目的 |
|--------|--------|-------|
| S-01 | ホーム / アップロード画面 | ユーザーが動画をアップロードする |
| S-02 | ステータス画面 | 編集処理の進行状況を確認する |
| S-03 | 完成動画画面 | 完成動画の表示・ダウンロード |
| S-04 | タイトル案画面（任意） | 自動生成されたタイトル候補を表示 |

---

# 📱 2. 詳細ワイヤーフレーム

---

## 👉 **S-01：ホーム / アップロード画面**

### 目的
- ユーザーが動画をアップロードし、自動編集プロセスをスタートさせる。

### 主なUI要素
- アップロードエリア（ドラッグ&ドロップ＋ボタン）
- 動画のプレビュー（任意）
- オプション入力
  - platform_hint（ドロップダウン）
  - max_duration_seconds
- 「自動編集を開始」ボタン

### 簡易ワイヤー

```
+--------------------------------------+
|       Auto Video AI Agent            |
+--------------------------------------+

    [ Select or Drop Video ]  ← アップロード領域

Platform:
[ TikTok | Shorts | Reels | None ]

Max Length:
[ 60 ] seconds

         [ Start Auto Editing ]  ← ボタン
```

### 遷移
アップロード完了 → video_id を取得 → S-02 に遷移

---

## 👉 **S-02：ステータス画面（処理中）**

### 目的
- Whisper → カット → テロップ → BGM の進行状況を表示する
- 完了後は S-03 に遷移

### UI要素
- video_id 表示
- ステータス（queued / processing / completed）
- プログレスバー（%）
- 処理ステップ一覧（チェックマーク付き）

### 簡易ワイヤー

```
+--------------------------------------+
|     Processing Video...  (45%)       |
+--------------------------------------+

Steps:
  ✓ Upload
  ✓ Transcribe (Whisper)
  → Cut Silence
    Subtitle
    BGM
    Export

[Refreshing every 2s...]
```

### 遷移
status = completed → S-03へ
status = failed → エラー表示

---

## 👉 **S-03：完成動画画面**

### 目的
- 完成動画を確認、ダウンロードできるようにする

### UI要素
- 動画プレビュー（埋め込み）
- ダウンロードボタン
- サムネイル確認
- タイトル案生成への導線（ボタン）

### 簡易ワイヤー

```
+--------------------------------------+
|        Your Video is Ready!          |
+--------------------------------------+

      [ Video Preview ]  ← 完成動画

      [ Download Video ]  ← mp4

Thumbnail:
      [ thumbnail.jpg ]

   [ Generate Title Suggestions ]  → S-04へ
```

---

## 👉 **S-04：タイトル案画面**

### 目的
- Whisperの文字起こし結果を基に SNS向けタイトルをAIが生成する

### UI要素
- タイトル候補表示（3〜5個）
- コピー機能（クリップボード）
- 再生成ボタン（任意パラメータあり）

### 簡易ワイヤー

```
+--------------------------------------+
|      Title Suggestions (AI)          |
+--------------------------------------+

【神ツール】編集ゼロで動画が完成するAIを作った
                                 [ Copy ]

忙しい社会人でも毎日投稿できる自動編集AIの話
                                 [ Copy ]

編集ゼロでショート動画量産できる時代が来た
                                 [ Copy ]

            [ Regenerate ]
```

# 🔗 3. 画面遷移図（シンプル版）

```
S-01（アップロード）
       ↓ video_id
S-02（処理ステータス）
       ↓ completed
S-03（完成動画）
       ↓ ボタン
S-04（タイトル案）
```

---

# 🎯 4. MVPに必要なフロント側ロジック

### S-01
- ファイル選択 → `/videos` にPOST
- レスポンスの `video_id` で S-02 に遷移

### S-02
- 2秒ごとに `/videos/{id}/status` をPoll
- 完成したら S-03へ

### S-03
- `/videos/{id}/result` を取得して表示

### S-04
- `/videos/{id}/title` をGETして表示
